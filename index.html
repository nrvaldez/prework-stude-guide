HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • h1, h2, h3, represents level of headings
  • p element represents a paragraph or a block of text
  • ul, ol, & li elements represents unordered list, ordered list, & list items
  • img element contains information about images that are displayed on the webpage

CSS

  • margin = same as margin in word processing. It indicates how much space we want around outside of element. set at 0 indicates no extra space around element.
  • padding = adds space around content inside an element. increase padding makes entire element bigger.
  • border = assigns a border with width.
  • rel = specifies relationship between current doccument and linked doc. resources.
  • href = specifies location (URL) of external resouces.
  • card = designates class selector any HTML assigned to card wil gain property assignments from CSS.
  • box shadow = adds shadow effects around an element frame. can set multiple effects with commas. runs on X (to right of the box) and Y (just below the box) offsets.
  • CSS file linked within index.html file within the header after meta before title.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout main: to switch to main branch
  • git commit -m: to commit file with notes
  • git add: to add files you just changed
  • git merge -x: to merge both branches
  • git pull origin main: to update local and remote main branch.
  • touch: adds in your new css (style.css) or JavaScript (script.js) these go into your assets folder
  • git add -A = adds all your changes in git

JavaScript

  • var = declares a variable give var a name (var shape)
  • =: to assign specific data to store in our variable example (var shape = "circle")
  • topics = string of topics we are studying
  • each line of Java ends with a semi colon ;
  • console.log = ouput a message example (console.log will output the value of topics)
  • array = single variable used to hold a group of data example (var shapes = ["triangles", "square", "pentagon", "circle"];) array holds 4 different shapes (data)
  • strings need to be contained with quotation marks ""
  • function = set of instructions that tells a computer how to perform a certain task
  • for = is used to create loops
  • control flow is the order in which a computer executes code in a script
